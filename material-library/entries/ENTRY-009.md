## 💎 金句摘录

1. "六运小区95%都是'住改商'，而新的热食证去年就不再允许办理。"

2. "天塌了，一口热乎的都吃不上了。"

3. "凭借天河CBD优越地段，六运小区聚集咖啡店、中古店、日料店、小酒馆等超百家特色商铺，成为广州顶流打卡点。"

4. "50平方米的商铺月租约22000元，70平方米月租约25000元，再加上2万-20万元不等转手费，妥妥的广州小区商铺租金天花板。"

5. "随着本轮餐厅、娱乐、洗浴等店铺的清出，未来，六运商业格局也会重新洗牌，取而代之的，或许是服装、文创小店这类无油烟、轻干扰业态。"

## 🔄 转化弧

- **读者进入时**：好奇/焦虑（听说要清退？还能去吗？）
- **中间认知转变**：了解到这是"住改商"政策收紧 + 安全隐患 + 居民投诉的综合结果
- **离开时情绪**：惋惜但理解，对未来商业格局变化有预期

## 🧠 心理模式 & 悖论

| 类型 | 内容 |
|------|------|
| **历史情怀悖论** | 小区从1987年全运会运动员村演变为网红打卡地，情怀与商业的张力 |
| **繁华与隐患并存** | 表面热闹（排队、打卡），背后是火灾、噪音、油烟投诉不断 |
| **政策传导延迟** | 政策6月1日执行，但商户们4月就在陆续撤退，市场比政策跑得更快 |
| **核心矛盾** | 居民想要安静生活 vs 商户想要流量收益 vs 监管想要安全合规 |

## 🧱 创意积木（可复用）

| 积木类型 | 提炼内容 | 通用公式 |
|----------|----------|----------|
| **反差开头法** | 现场热闹 vs 背后暗流涌动 | "即便是工作日，人流量依旧不低...然而，热闹表象下是暗流涌动" |
| **数据锚定法** | 用具体租金数据强化"顶流"标签 | "50平22000元/月 + 2-20万转让费 = 广州商铺天花板" |
| **历史溯源法** | 从小区1987年起源讲起，赋予地点故事感 | "1987年第六届全国运动会运动员村 → 1989年改建居民小区 → 如今网红打卡地" |
| **多方声音法** | 居委会说法 vs 商户说法 vs 网友哀嚎 | 让不同利益方自己说话，读者自己判断 |
| **未来预测收束** | 结尾给出一个理性预期 | "取而代之的，或许是...这类无油烟、轻干扰业态" |
