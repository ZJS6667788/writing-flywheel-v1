📚 写作飞轮

飞书表格使用指南 - 让爆款拆解与创意生成形成闭环

📊 表格结构总览

📂 飞书多维表格 ├── 📚 素材库（ENTRY） │ └── 爆款文章拆解 + 原始素材沉淀 │ ├── 💡 创意方向库（IDEA） │ └── 独立管理的写作方向卡片 │ ├── 📝 场景1完稿（爆款延伸） │ └── 基于爆款拆解后的延伸写作 │ └── ✍️ 场景2完稿（原创首发） └── 原创灵感 + 创意方向 → 新文章

一、素材库（ENTRY）

用于收集和拆解爆款文章，沉淀可复用的创意。

字段说明

字段说明
编号ENTRY-XXX 格式，如 ENTRY-001
标题爆款文章标题
来源链接原文链接（转载）或灵感来源
拆解结构文章的行文框架（开头→推进→结尾）
金句触动你的句子
创意积木可复用的写作模块
创意方向从本文提取的可迁移场景方向
完稿内容对应的完稿正文
单选转载 / 原创
日期入库时间

使用流程

1发现爆款 → 入库

复制微信公众号链接，在「素材库」新增记录，填入编号、标题、来源链接

2拆解爆款

完善「拆解结构」「金句」「创意积木」

3提取创意方向

在「创意方向」字段，写出可迁移的场景

二、创意方向库（IDEA）

💡 这是写作飞轮的核心！
每个 IDEA 是一张独立的写作卡片，关联参考素材，设置优先级，追踪写作状态。

字段说明

字段说明
编号IDEA-XXX 格式，如 IDEA-001
推荐标题这个方向用什么标题
核心角度独特切入视角
目标读者读者画像
核心论点一句话主张
创意积木本次要用的技巧组合
参考素材关联的 ENTRY 编号/链接
优先级★★★ 高★★☆ 中★☆☆ 低
状态待写作 / 写作中 / 已完成

使用流程

1生成创意方向

从「素材库」选中 1-3 条相关素材，基于「创意方向」字段生成新的 IDEA 卡片

2关联参考素材

在「参考素材」填入关联的 ENTRY 编号

3设置优先级

★★★ 高立刻要写   ★★☆ 中本周要写   ★☆☆ 低以后再说

4追踪状态

待写作 → 写作中 → 已完成

视图建议

按优先级分组：快速看到哪些要优先写
按状态分组：看到当前写作进度
全部卡片：总览所有创意方向

三、场景1完稿（爆款延伸）

基于已有爆款的「创意方向」延伸写作。

字段说明
编号DRAFT-S1-XXX，如 DRAFT-S1-001
标题新文章标题
关联素材对应的 ENTRY 编号/链接
参考创意引用的 IDEA 编号
完稿正文完整文章
发布日期发布后填
发布平台公众号 / 小红书 / 微博 / 知乎 / 其他

四、场景2完稿（原创首发）

新的灵感/素材 + 借鉴创意方向 → 原创文章。

字段说明
编号DRAFT-O-XXX，如 DRAFT-O-001
标题新文章标题
素材来源灵感来源
使用的创意借鉴的爆款创意编号
完稿正文完整文章
发布日期发布后填
发布平台公众号 / 小红书 / 微博 / 知乎 / 其他

五、完整工作流示例

场景1：爆款拆解 → 完稿

1发现爆款文章
2「素材库」新增 ENTRY-009
3拆解爆款：填拆解结构、金句、创意积木
4提取创意方向
5「创意方向库」新建 IDEA-005
6执行写作
7「场景1完稿」新建记录
8发布后填日期和平台

场景2：灵感 → 原创文章

1有了灵感/发现新素材
2「素材库」新增 ENTRY-010（单选=原创）
3查看「创意方向库」，找可借鉴的 IDEA
4「创意方向库」新建 IDEA-006
5执行写作
6「场景2完稿」新建记录
7发布
