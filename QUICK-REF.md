## 三个命令详解

### 命令 1：/CONTENT-GOLDMINE（拆解入库）

**用法：**
```
/CONTENT-GOLDMINE https://mp.weixin.qq.com/s/xxxxx
```
或者直接粘贴文章正文。

**效果：**
- 自动拆解：核心结构 + 金句 + 心理模式 + 创意积木
- 存入素材库，生成唯一ID（ENTRY-XXX）
- 输出可视化报告

**快捷版：**
```
/CG + 链接或内容
```

---

### 命令 2：/IDEA-SPARK（生成创意）

**触发条件：** 素材库有 1 条以上内容

**用法：**
```
/IDEA-SPARK
```

**效果：**
- 扫描所有素材条目
- 提炼今天可以写的 3-5 个创意方向
- 每个方向附带：标题、核心结构、开头句、素材来源

**何时用：**
- 每天写稿前
- 不知道写什么的时候
- 想找新角度的时候

**快捷版：**
```
/IS
```

---

### 命令 3：/WRITING-CLONE（自动写作）

**用法：**
```
/WRITING-CLONE 1
```
（1 是 /IDEA-SPARK 输出的方向编号）

或者：
```
/WRITING-CLONE [你自己想的方向描述]
```

**效果：**
1. 自动从素材库采集相关创意积木
2. 生成完整初稿
3. **自动把新文章拆解后反哺素材库** ← 飞轮闭环！

**快捷版：**
```
/WC 1
```

## 常见问题

**Q：每次拆解要花多久？**
A：系统自动处理，30秒内完成。

**Q：素材库会越来越大吗？**
A：会的，但这是好事——写得越多，素材越多，下次写作越轻松。

**Q：飞轮是什么意思？**
A：每写一篇文章 → 拆解后反哺素材库 → 素材库更丰富 → 下次写作质量更高 → 飞轮越转越快。

**Q：可以删除素材条目吗？**
A：可以，直接删除 `material-library/entries/ENTRY-XXX.md` 文件即可。

## 创作方法论来源

本系统基于 **Dan Koe 写作方法论**：
> *"写作不是从空白开始，而是站在之前所有积累上。"*

三个核心思想：
1. **输入决定输出** — 先学会拆解好文章
2. **积木思维** — 把文章拆成可复用的创意积木
3. **飞轮效应** — 写得越多，积累越多，写得越好
